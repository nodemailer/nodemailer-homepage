# Guides

Step-by-step guides for common email sending scenarios.

- **[Testing with Ethereal](/guides/testing-with-ethereal)** - Use Ethereal.email to test email sending without delivering to real recipients. Generate test accounts automatically and preview sent messages in a web interface.

- **[Using Gmail](/guides/using-gmail)** - Send email through Gmail using OAuth2 or App Passwords. Covers authentication methods, sending limits, and common issues.
